# Bundle Builder

## Tech Stack

- React
- TypeScript
- Tailwind
- Zustand
- Vite

## Features

- Multi-step accordion
- Variant-based quantity tracking
- Live review panel
- Dynamic pricing
- Local storage persistence
- Responsive design

## Run

npm install

npm run dev

## Architecture

Feature-based architecture

State managed with Zustand

Data-driven UI using JSON configuration
