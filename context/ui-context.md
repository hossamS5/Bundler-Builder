# UI Context

The UI should closely match the provided Figma.

---

## Layout

Desktop:

Builder | Review Panel

Two-column layout.

Builder occupies approximately 70%.

Review panel occupies approximately 30%.

---

## Mobile

Review panel moves below builder.

Accordion remains usable.

No horizontal scrolling.

---

## Product Card

States:

### Default

Neutral border.

### Selected

Highlighted border.

### Hover

Subtle border emphasis.

---

## Quantity Stepper

Must support:

- decrement
- increment

Prevent negative values.

Disable minus button at zero.

---

## Accordion

Open step:

- highlighted
- expanded

Closed step:

- collapsed

---

## Review Panel

Sticky on desktop.

Scrollable if content exceeds viewport.

---

## Accessibility

Buttons must be keyboard accessible.

Inputs must have labels.

Interactive elements must have focus states.
